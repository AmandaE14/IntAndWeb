1. Classes are templates for creating objects.
2. The visibility of a property, a method or a constant can be defined by prefixing the declaration with the keywords public, protected or private.
3. When a class derives from another class it is called inheritance.
4. Namespaces allow the PHP interpreter to tell the difference between two classes, functions, or constants that share the same name.
5. Several predefined variables in PHP are superglobals, which means they are available in all scopes throughout a script.
6. Functions can be called directly, from within a script, to perform a specific task.
7. The header() function sends a raw HTTP header to a client.
8. MVC is an architectural/design pattern that separates an application into three main logical components Model, View, and Controller.
9. Libraries allow developers to use code that they, or other programmers, have already written to perform a task.
