1. HTML stands for HyperText Markup Language. It is the standard language for making websites. It organizes the contents of a webpage.
2. CSS stands for Cascading Style Sheets. It is used to design web pages. It determines the colors, fonts, spacing, and layout.
3. The Fordham website features a header for navigation, main sections for important information and a footer with more links. The layout is straightforward, making it easy for users find information.
4. An HTML element is a piece of content in a webpage. A tag is the code that defines the element. An attribute provides additional information about the element.
5. A CSS rule is made up of a selector and a declaration block that includes features like color and size.
6. CSS rules cascade, which means that the order of the rules counts. If many rules apply to the same element, the latest or most specific rule takes priority.
